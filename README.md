# 成祖燕窩 Cheng Tsu Bird's Nest

一個單頁式靜態網站，展示燕窩品牌故事、產品（乾燕盞）、聯絡方式，並支援中英文切換，使用 Bootstrap 5 進行響應式設計。

## 技術

- HTML5
- CSS3
- JavaScript
- Bootstrap 5 CDN
- Font Awesome (FB icon)

## 主要頁面文件

- `index.html`：網頁結構與內容
- `style.css`：自定義樣式
- `script.js`：語系切換、平滑捲動邏輯

## 使用方式

1. 開啟 `index.html` 即可預覽。
2. 點選右上角語系切換按鈕切換中/英。
3. 點選導覽列錨點可平滑移動到對應區塊。

## Assets

- `assets/ChengTsu.png`：品牌 logo
- `assets/BirdNest_01.jpg`、`assets/BirdNest_02.jpg`：產品照片
